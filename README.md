# ゲームの始め方
1. git clone `https://github.com/yuukii125/Porker.git`<br>
2. cd PORKER/<br>
3. ruby porker.rb<br>
4. 自分の手札を見て、勝負するなら1を、降りるなら3を入力してください<br>
5. 勝敗が決まります<br>
6. もう一度対戦するなら1を、やめるなら3を入力してください<br>

## ゲーム内容
* CPとあなたに手札が5枚ずつ配られます
* 手札交換はないので、その手札で勝負するか降りるか選択します
* 降りた場合はあなたの負けです
* フラッシュ系の役以外で勝負判定します
* 数字の大小で勝敗は決まりません
